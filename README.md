🚀 Sobre el proyecto
Portfolio personal construido con HTML · CSS · JavaScript vanilla. Sin frameworks, sin dependencias. Diseño editorial oscuro inspirado en portfolios de diseñadores creativos top, con identidad cordobesa y actitud propia.

✨ Features
→ Ticker / marquee animado
→ Hero tipográfico fullscreen
→ Grid de proyectos con hover
→ Stack visual como pills/tags
→ Scroll reveal con IntersectionObserver
→ Cursor personalizado con lerp
→ Nav sticky con blur
→ 100% responsive mobile-first
